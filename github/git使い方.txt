git cloneでファイルをコピーする

% git status

で現状が確認できます．もし，ファイルを追加，書き換えた場合は，

% git add filename

などとして，変更を更新する対象として追加してください．その後，

% git commit -m "comment"

で，コメント付きで(ローカル)レポジトリを更新します．Dropbox
以下のマスターレポジトリを変更して，我々が見えるようにするに
は，
% git push origin master
としてください．

仮に私が書き換えて，大高くんにその内容を持って行って欲しい場
合は，
% git pull
としてもらうことになります．
